# test-bear

Запускать c помощью "test bear.exe".

Перемещение по плоскости осуществляется с помощью WASD. Эта плоскость не пересекает пространство, в которой перемещаются фигуры. Это позволяет сохранить вид сверху.

При зажатии Left CTRL есть возможность выделить фигуры, кликнув по ним указателем. Вместе с этим появляется окошко с названием и координатами фигур.

Выход из приложения осуществляется клавишей ESC.
